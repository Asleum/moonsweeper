Class MouseInputsManager

Static class for registering and reacting to mouse inputs

Fields

MouseInputsManager.LeftClickHandlers Dictionary that associates a Roblox part with a function to be executed once the part is clicked
MouseInputsManager.RightClickHandlers Same as above, but for right clicks
MouseInputsManager.Mouse Reference to the player's mouse
MouseInputsManager.SelectionBox A Roblox SelectionBox that is shown over any hovered activable part

Methods

MouseInputsManager\Initialize () Initialize fields and start reacting to clicks
MouseInputsManager\BindPartToClick (part, leftClickCallback, rightClickCallback) Set some functions to be executed once a part is left or right clicked
MouseInputsManager\UnbindPartFromClick (part) Make the given part not react to clicks anymore
MouseInputsManager\UpdateSelectionBox () Update the selection box's adornee to be the hovered clickable part, if applicable
